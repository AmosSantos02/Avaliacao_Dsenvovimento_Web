Projetos

🎲 AluGames

Descrição

Sistema de gerenciamento de aluguel de boardgames. O usuário pode alternar o status de cada jogo entre "disponível" e "alugado", com atualização visual automática do botão e da imagem. Desenvolvido com manipulação de DOM e lógica de estados em JavaScript.

🎁 Amigo Secreto

Descrição

Aplicação para organizar sorteios de amigo secreto. O usuário adiciona participantes, e o sistema embaralha a lista usando o algoritmo de Fisher-Yates para gerar os pares de forma aleatória e justa. Conta com validações para evitar nomes duplicados e repetição de sorteio.

Tecnologias

Github

🛒 Carrinho de Compras

Descrição

Simulação de um carrinho de compras interativo. O usuário seleciona produtos, define a quantidade e o sistema calcula automaticamente o subtotal e o valor total da compra em tempo real, com opção de limpar o carrinho.

Tecnologias

Github

🎟️ Compra de Ingressos

Descrição

Sistema de venda de ingressos com controle de estoque em tempo real. O usuário escolhe o tipo de ingresso (pista, cadeira superior ou inferior) e a quantidade, e o sistema verifica a disponibilidade e atualiza o estoque automaticamente após cada compra.
